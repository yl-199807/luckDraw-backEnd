# luckDraw-backend

> 一个仿掘金的抽奖系统项目——后端部分
>
> （因为在导师这里实习紧张，node后端部分只做了查询奖品的接口，登录接口还没做完）

| 版本    | 描述                                            | 时间     |
| ------- | ----------------------------------------------- | -------- |
| v 1.0.0 | 基于express框架的node后端服务，完成查询奖品接口 | 2021-9-5 |

## 环境及版本说明

* node 版本：12.19.0
* express 版本：4.17.1
* nodemon 版本：2.0.12

## 项目配置

``` bash
# install dependencies
npm install

# run for production
node server.js

# run 
nodemon server.js
```

## 代码说明

* server.js 接口文件

* prize.json 奖品数据
